FAQ

• Comment avez-vous décomposé le projet ? Quelles ont été les phases clés


J'ai décomposé le projet en plusieurs phases :

1. Phase de test de l'API avec Postman

2. Phase de développement du backend en Go

3. Phase de développement du frontend en HTML/CSS

4. Phase de tests et de débogage

5. Phase de documentation


• Comment avez-vous réparti les tâches entre les membres de l'équipe ? Avezvous utilisé une stratégie particulière pour organiser le travail ?


J'ai découpé le backend en plusieurs parties : contrôleurs, modèles, routes, etc.

Et j'ai avancer de features en features jusqu'à la fin du projet.

• Comment avez-vous géré votre temps ? Avez-vous défini des priorités ?


J'ai travaillé la plus part du temps dans les heures de cours, mais j'ai aussi travaillé pendant les week-ends pour débugger et finaliser le projet.


• Quelle stratégie avez-vous adoptée pour vous documenter ?


J'ai utilisé la documentation officielle de Go, des video tutoriels sur YouTube, et j'ai aussi utilisé des forums comme StackOverflow et lorsque j'ai eu des difficultés, j'ai demandé des explication à l'IA

J'ai cité mes sources sur les morceaux de code utilisés dans le projet.
